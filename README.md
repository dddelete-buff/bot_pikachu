# bot_pikachu
Это учебный проект "Телеграм бот Пикачу". 
Бот написан на Python с использованием библиотеки python-telegram-bot.
Реализованы основные команды: /start (приветствие), /help (справка), /pikachu (отправка случайного изображения)
Изображения разбиты по эмоциональным категориям (happy, cute, angry, sleepy). 
Файлы изображений хранятся локально в папке pikachu/, загрузка происходит через прямое чтение из директории.
